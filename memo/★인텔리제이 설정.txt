▶ 초기 설정

	Auto import (자동 임폴트)
		자동 임포트 옵션 활성화

	File Encoding (파일 인코딩)
		3개 전부 UTF-8 로 통일

	Code Completion (코드 완성)
		대소문자를 구분할지 여부 체크
		체크를 하지 않아야 대소문자 구분하지 않음.
		맨 위에 있는 체크 박스 체크

	Annotation processor (어노테이션 프로세서)
		주석 처리기 사용 활성화


▶ 단축키들 

	Ctrl + D : 한줄 복사 
	Ctrl + X : 한줄 삭제 (잘라내기/Ctrl + V로 붙혀넣기도 가능하다.)
	Shift + F6 : 이름 변경 (변수, 메소드 등등 이름을 변경이 가능하다.)
	Ctrl + Alt + L : 자동 줄 맞춤
	Ctrl + shift + F : 검색기능
	Ctrl + shift + R : 단어 변경 기능
